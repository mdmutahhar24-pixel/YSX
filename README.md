## Features:
YSX includes many features such as:
- a home page with a warm welcome
- a feed page to watch you favorite Reels
- Reels that contain comments, likes, tags, creator recognition, and so much more!
- a create button to create your own post
- a chat page to have one on one conversations with people (no group chats)
- Settings page to either delete or log off your account
- Account page to fully customize yourself
- Admin features to prevent inappropriate content
- a ban system with a ban pate that states ban expiry and ban reason
- a light and dark mode
- fully mobile compatible

## Security Requirements for account password:
- Needs to have: a capital letter, a number, a special character, lowercase letters, at least 11 characters
